### 接口文档

#### 认证模块
1. **登录**  
   - URI: `POST /auth/login`  
   - 请求结构:
     ```json
     {
       "username": "string",
       "password": "string"
     }
     ```
   - 成功响应结构 (200):
     ```json
     {
       "user_id": "int",
       "username": "string",
       "email": "string",
       "avatar_url": "string",
       "role": "string",
       "last_active": "string"
     }
     ```
     - Header: `Authorization: Bearer <token>`

2. **发送验证邮件**  
   - URI: `POST /auth/verify`  
   - 请求结构:
     ```json
     {
       "email": "string"
     }
     ```
   - 成功响应: `邮件已发送至{email}` (200)

3. **注册**  
   - URI: `POST /auth/register`  
   - 请求结构:
     ```json
     {
       "username": "string",
       "email": "string",
       "password": "string",  // 加密后的密码
       "vrfcode": "string"    // 邮件验证码
     }
     ```
   - 成功响应结构 (200):
     ```json
     {
       "user_id": "uint",
       "username": "string"
     }
     ```

4. **保持登录获取用户信息**  
   - URI: `GET /auth/my_info`  
   - 成功响应结构 (200):
     ```json
     {
       "user_id": "uint",
       "username": "string",
       "email": "string",
       "avatar_url": "string",
       "role": "string",
       "last_active": "string"
     }

#### 用户模块
1. **获取用户信息**  
   - URI: `GET /api/user/{id}`  
   - 成功响应结构 (200):
     ```json
     {
       "user_id": "uint",
       "username": "string",
       "email": "string",
       "avatar_url": "string",
       "role": "string",
       "last_active": "string"
     }
     ```

2. **获取用户列表** (管理员权限)

   - 成功响应结构 (200):

     [
       {
         "user_id": "uint",
         "username": "string",
         "email": "string",
         "avatar_url": "string",
         "role": "string",
         "last_active": "string"
       },
       ...
     ]
     ```

3. **用户活跃心跳**  
   - URI: `GET /api/user/ping`  
   - 成功响应: `pong` (200)

#### 社团模块
1. **获取社团列表**  
   - URI: `GET /api/club/list?offset=0&num=10`  
   - 成功响应结构 (200):
     ```json
     [
       {
         "club_id": "int",
         "club_name": "string",
         "category": "int",
         "tags": ["string"],
         "logo_url": "string",
         "desc": "string",
         "created_at": "string",
         "member_count": "int"
       },
       ...
     ]
     ```

2. **获取社团详情**  
   - URI: `GET /api/club/{id}/info?post_num=5`  
   - 成功响应结构 (200):
     ```json
     {
       "club_id": "int",
       "club_name": "string",
       "category": "int",
       "logo_url": "string",
       "desc": "string",
       "created_at": "string",
       "member_count": "int",
       "members": [
         {
           "member_id": "int",
           "user_id": "int",
           "club_id": "int",
           "role_in_club": "string",
           "joined_at": "string",
           "last_active": "string"
         }
       ],
       "posts": [
         {
           "post_id": "int",
           "club_id": "int",
           "author_id": "int",
           "title": "string",
           "comment_count": "int",
           "created_at": "string"
         }
       ]

     ```

3. **申请加入社团**  
   - URI: `POST /api/club/{id}/join`  
   - 请求结构：
     ```json
     {
        "reason": "string"
     }
     ```
   - 成功响应: `加入社团申请成功` (200)

4. **申请创建社团**  
   - URI: `POST /api/club/create`  
   - 请求结构:
     ```json
     {
       "name": "string",
       "desc": "string",
       "requirements": "string"
     }
     ```
   - 成功响应: `创建社团申请成功` (200)

5. **收藏社团**  
   - URI: `POST /api/club/favorite`  
   - 请求结构:
     ```json
     {
       "club_id": "string"
     }
     ```
   - 成功响应: `收藏社团成功` (200)

6. **取消收藏社团**  
   - URI: `POST /api/club/unfavorite`  
   - 请求结构:
     ```json
     {
       "club_id": "string"
     }
     ```
   - 成功响应: `取消收藏社团成功` (200)

7. **获取社团总数**  
   - URI: `GET /api/club/club_num`  
   - 成功响应结构 (200):
     ```json

       "club_num": "number"
     }
     ```
8. **获取社团类别列表**  
   - URI: `GET /api/club/catogories`  
   - 成功响应结构 (200):

       ...

9. **获取我的社团列表**

#### 帖子模块
1. **获取社团帖子列表**  
   - URI: `GET /api/club/post/posts/{club_id}?offset=0&post_num=10`  
   - 成功响应结构 (200):
     ```json
     [
       {
         "post_id": "int",
         "club_id": "int",
         "author_id": "int",
         "title": "string",
         "comment_count": "int",
         "created_at": "string",
         "content_url": "string"
       }
     ]
     ```

2. **获取置顶帖**  
   - URI: `GET /api/club/post/pinned/{club_id}`  
   - 成功响应结构 (200):
     ```json
     {
       "post_id": "int",
       "club_id": "int",
       "author_id": "int",
       "title": "string",
       "comment_count": "int",
       "created_at": "string"
     }
     ```

3. **创建帖子**  
   - URI: `POST /api/club/post/create`  
   - 请求格式: `multipart/form-data`  
   - 参数:
     - `title`: 帖子标题
     - `club_id`: 社团ID
     - `content`: 帖子内容
   - 成功响应: (创建成功无返回体，200)

4. **获取帖子实际内容**  
   - URI: `GET /pub/post_files/{id:int}`  
   - 成功响应结构 (200):
     Markdown 文件

---

### 权限说明
- **管理员接口** 需要 Header: `Authorization: Bearer <token>` 且用户角色为 `admin`
- **发布者接口** 需要角色为 `publisher` 或 `admin`
- 所有接口均需身份验证（除认证模块）
